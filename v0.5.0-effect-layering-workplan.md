# v0.5.0 Effect Layering Workplan

## 本轮目标

把 `v0.5.0-card-presentation-pass` 从“统一卡牌表现底盘已成型”，推进到“不同效果类型真的有不同手感”。

重点不是再加更多卡，而是让以下几类效果在视觉和节奏上真正分层：

- 摸牌
- 弃牌
- 回收
- 夺取
- 冻结
- 销毁
- 公共牌操作

---

## 当前进度（第一批）

已完成第一批底盘与代表技能接入：

- 新增 `effect_layering_lab` 调试场景，可手动 / 自动复验不同结果去向
- 已能分层预览并在真实流程里接入的代表技能：
  - 囤积：`强行补给`、`提纯`、`筛选`、`战术武装`、`威压`
  - 潮汐：`降雨`、`声纳`、`寒潮`、`冲刷`、`间歇泉`、`潜流`
  - 秩序：`紧急征收`、`资产查没`、`内幕交易`、`强制征用`、`暗箱操作`
  - 枪手：`战术校准`、`快速装填`、`幸运币`、`致命判定`、`穿甲弹`、`威慑射击`、`火力压制`
  - 掘墓：`亡者低语`、`骨矛`、`鲜血交易`、`疯狂挖掘`、`尸骸拼凑`、`冥界之门`
- 当前已分层的结果类型：
  - `draw` 牌库 → 手牌 / 公共牌
  - `discard` 来源区 → 弃牌堆
  - `recover` 弃牌堆 → 手牌 / 公共牌
  - `steal` 目标玩家 → 发动者
  - `freeze` 公共牌冻结与冻结脉冲
  - `destroy` 公共牌破坏 / 移除
  - `public` 公共牌新增 / 替换 / 补发
  - `swap` 自身手牌 ↔ 公共牌秘密互换
  - `peek` 目标区域高亮 + 结果收益
  - `skillGain` 弃牌后获得技能卡
  - `pressure` 手牌差距 / 盖牌冻结带来的目标压制

仍待后续扩展：

- 更多技能的真实流程挂点
- 销毁 / 冻结 / 公共池替换的更细分视觉差异
- 运行时与旧 helper 动效的进一步去重
- 被动 / 共鸣触发后的结果去向继续扩到更多真实分支

## 时间盒

总时长：`3 ~ 5 小时`

建议拆成 5 段：

1. 效果类型梳理与映射：`30 ~ 40 分钟`
2. 基础表现层接入：`70 ~ 90 分钟`
3. 代表技能接入与真实流程挂点：`70 ~ 90 分钟`
4. 测试场景与调试入口：`40 ~ 60 分钟`

## 具体任务

### 1. 效果类型映射梳理

先建立一份稳定映射，明确每类效果该走什么表现：

- `draw`
  - 从牌库飞向手牌区
- `discard`
  - 从来源区飞向弃牌堆
- `retrieve`
  - 从弃牌堆飞回目标区
- `steal`
  - 从目标玩家区域抽离资源 / 卡牌，再飞向发动者
- `freeze`
  - 目标卡牌结霜、高亮、冻结标记
- `destroy`
  - 目标卡牌碎裂或黑雾散去
- `public`
  - 公共牌区新增、替换、移除、补发

产出：

- 一份稳定的 effect type -> motion grammar 对照
- 明确哪些技能属于单类型，哪些属于复合类型

### 2. 基础表现层接入

在现有 `Card FX Overlay` 上补齐“命中后的具体去向”：

- 技能成卡后，不再只有中间展示
- 结果阶段真正飞向对应区域
- 数值变化与卡牌移动同步

首批必须接通：

- `fxDrawCards(...)`
- `fxDiscardCards(...)`
- `fxRecoverFromDiscard(...)`
- 公共牌新增 / 替换 / 移除的飞牌封装

目标：

- 让“摸牌像摸牌”
- “弃牌像弃牌”
- “回收像从弃牌堆拿回”

### 3. 代表技能接入

优先接这批技能，覆盖主要效果类型：

- 囤积
  - `强行补给`
  - `提纯`
  - `筛选`
- 潮汐
  - `降雨`
  - `寒潮`
  - `冲刷`
- 秩序
  - `紧急征收`
  - `资产查没`
  - `强制征用`
- 枪手
  - `快速装填`
  - `穿甲弹`
  - `威慑射击`
- 掘墓
  - `亡者低语`
  - `骨矛`
  - `冥界之门`

要求：

- 代表技能的“类型分层”必须在真实流程里可见
- 不只在测试台里好看，实战里也要成立

### 4. 测试场景补强

在现有调试系统上补一组更聚焦的入口：

- `effect_layering_lab`
  - 专门检查摸牌 / 弃牌 / 回收 / 销毁 / 公共牌替换

场景应支持：

- `manual`
  - 点按钮逐项播放单类效果
- `auto`
  - 自动轮播主要效果类型，快速看差异是否成立

必须覆盖：

- 牌库 → 手牌
- 手牌 / 目标区 → 弃牌堆
- 弃牌堆 → 手牌 / 公共牌区
- 目标卡冻结
- 目标卡销毁
- 公共牌替换 / 补发

### 5. 回归与总结

完成后要做一轮最小回归：

- `demo.html` 语法检查通过
- `卡牌表现测试台` 正常
- `文案与注释测试台` 不受影响
- `陷阱 / 被动 / 共鸣` 的现有效果不被这轮带坏
- 至少打一轮真实流程，确认不会因为飞牌挂点导致卡死

最后要输出：

- 本工作块完成了什么
- 哪些效果类型已经真正分层
- 还有哪些效果只是第一版
- 下一轮最值得继续做什么

## 本轮验收标准

当以下条件成立时，本工作块可视为完成：

- 同一批技能不再只有“换卡面、换文案”，而是结果阶段真的有不同去向
- 玩家能通过动画直接分辨：
  - 摸牌
  - 弃牌
  - 回收
  - 冻结
  - 销毁
  - 公共牌替换
- 新测试场景可复验这些差异
- 既有技能成卡、停留、离场节奏不被破坏

---

## 明确不在本轮处理

以下内容本轮不作为主目标：

- hover 说明的最后一层排版精修
- 陷阱 / 被动 / 共鸣的进一步个性化气质差异
- 每一张卡都做单独定制动画
- 更大范围的规则平衡调整

---

## 风险提醒

- 飞牌挂点过多，容易把原有流程带卡
- 真实技能分支很多，可能出现“测试台正常，实战挂点不全”
- 公共牌 / 弃牌堆 / 手牌三类区域的 DOM 锚点不统一，可能需要补通用 helper

应对方式：

- 先做代表技能，不一口气全量铺开
- 每接完一类效果，就回到测试台和真实流程双验
- 不破坏现有主动技能成卡主链

---

## 当前已接入的代表技能

目前已经进入“结果阶段真实分层”的代表技能包括：

- 囤积：`强行补给 / 提纯 / 筛选 / 战术武装 / 堡垒武装`
- 潮汐：`降雨 / 声纳 / 冰封 / 间歇泉 / 潜流 / 寒潮 / 冲刷`
- 秩序：`紧急征收 / 资产查没 / 强制征用 / 火力压制 / 威压 / 加税令`
- 枪手：`快速装填 / 战术校准 / 穿甲弹 / 威慑射击 / 幸运币 / 致命判定 / 暗箱操作`
- 掘墓：`亡者低语 / 骨矛 / 鲜血交易 / 疯狂挖掘 / 尸骸拼凑 / 冥界之门 / 瘟疫`

这批技能已经不再只是共享统一成卡模板，而是在结果阶段具备以下分层手感：

- 牌库 → 手牌
- 手牌 / 目标区 → 弃牌堆
- 弃牌堆 → 手牌 / 公共牌区
- 目标卡冻结 / 盖牌压制
- 目标卡销毁 / 弃牌堆压制
- 公共牌新增 / 替换 / 回推
- 金币抽离 / 资源压制
- 技能卡获取 / 技能区收束

并且本轮又补上了一层“区域手感对齐”：

- 被动 / 共鸣结果分层第一批已经接通：
  - 被动：`帝国国库 / 严苛税法 / 水压充能 / 紧急变现 / 金权交易 / 鹰眼 / 亡灵天灾 / 左轮置换 / 战术口袋 / 黑暗仪式 / 尸体收益 / 赏金猎人`
  - 共鸣：`绝对容积 / 涨潮 / 黑金私军 / 走火 / 双发 / 虚空之手 / 亡者加护`
- 对应的代表结果方向现在已经能在真实触发里看到：
  - 金币获得 / 夺取 / 失去 / 支付
  - 检索后入手
  - 判定牌落弃
  - 弃牌堆回收
  - 额外盖牌
  - 公共牌补翻
  - 下回合奖励摸牌

- `筛选 / 快速装填 / 战术校准` 会更明确地把焦点收束到手牌区
- `降雨 / 声纳 / 冲刷 / 威慑射击` 会更明确地表现牌库到公共区的流动
- `强制征用 / 致命判定` 的偷牌会和普通摸牌区分成独立的夺取飞行
- `紧急征收 / 威压 / 火力压制 / 加税令 / 瘟疫` 会更明确地区分金币抽离、弃牌惩罚和光环压制
- `提纯 / 幸运币 / 冥界之门 / 鲜血交易 / 尸骸拼凑` 会更明确地区分净化、判定、回收、技能获取、尸骸回手这些结果阶段

测试台当前也已预置更多公共牌，以便直接验证：

- `降雨 / 威慑射击` 的新增公共牌
- `冲刷 / 冥界之门` 的替换与回收
- `降雨` 触发后可能出现的超上限弃置
- `间歇泉 / 潜流 / 寒潮` 对高密度公共区的反馈差异
